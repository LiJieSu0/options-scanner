# Options Scanner - 網站規格書

## 專案概述
- **名稱**: Options Scanner
- **類型**: 期權價格查詢工具網站
- **目標用戶**: 交易者、投資者
- **核心功能**: 查詢美股期權價格、篩選功能、記住篩選條件

## UI/UX 規格

### 配色方案 (深色主題)
- **背景**: #0f0f23 (深藍黑)
- **卡片背景**: #1a1a2e
- **主要按鈕**: #6366f1 (Indigo)
- **次要按鈕**: #22c55e (Green)
- **文字**: #e2e8f0 (淺灰白)
- **邊框**: #334155
- **紅色 (下跌)**: #ef4444
- **綠色 (上漲)**: #22c55e

### 排版
- **Header**: 固定頂部，Logo + 搜尋欄
- **主要區域**: 左側搜尋、中間表格、右側篩選
- **響應式**: 支援桌面和平板

### 元件
1. **搜尋框** - 輸入股票代碼
2. **期權表格** - 顯示履約價、Bid、Ask、成交量、IV
3. **篩選面板** - 履約價範圍、到期日、權利金範圍
4. **收藏列表** - 追蹤的股票
5. **篩選條件記憶** - localStorage 保存

## 功能規格

### 1. 搜尋功能
- 輸入股票代碼 (如 NVDA, TSLA, SOFI)
- 顯示現價和基本資訊
- 載入期權鏈數據

### 2. 期權鏈顯示
- 顯示 Put 和 Call 兩個區塊
- 欄位：履約價、Bid、Ask、中間價、成交量、IV
- 區分價內(ITM)和價外(OTM)

### 3. 篩選功能
- 履約價範圍 (min/max)
- 到期日選擇
- 權利金範圍
- IV 範圍
- 只看 OTM 或 ITM

### 4. 記住篩選 (localStorage)
- 保存最後使用的篩選條件
- 保存收藏的股票列表

### 5. 收藏功能
- 添加/移除股票到收藏
- 快速切換已收藏的股票

## 技術堆疊
- Next.js 14 (App Router)
- Tailwind CSS
- yfinance (期權數據)
- TypeScript
